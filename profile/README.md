# Orbier Group of Libraries and Co. 

## About Us!

Orbier Group of Libraries and Co. is a non-profit organization dedicated to building quality-of-life (QoL) tools, developer-friendly wrappers, and open technologies that simplify complex workflows.

Our mission is rooted in a simple belief:

Software should be **free, accessible, and empowering for everyone**!

## Our Core Principle: Free Software

We strongly believe in the philosophy of Free Software — software that respects users’ freedom to:

- Run the program for any purpose
- Study how it works
- Modify it
- Share it with others

Our projects aim to align with these principles first and foremost.

## Speciality about us 

We build the Following things - 
- Lightweight developer tools
- Wrappers For complex Systems
- Libraries
- Utilities that improve developers' efficiency

## Our Values <3

Beyond software, we care about real-world impact:

### Gender Equality & Women’s Empowerment

We actively support initiatives that promote equal opportunities, rights, and safety for women, especially within Developing Countries Like India.
Together, men and women, will build the foundation for our countries!

### LGBTQ+ Inclusion

We support a welcoming and respectful environment for people of all identities, including LGBT! Without them, we will lose economical value!
Inclusion is not optional — it’s foundational! 

## Non-Profit Commitment

Orbier operates as a non-profit initiative.

Any funds and support that is generated will mostly go towards NPO and NGOs that support Women Equality, especially in India.

We aim to combine technology with meaningful contribution.
