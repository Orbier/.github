# How To contribute? 

You can contribute in every ways, through reporting issues or pull requests!
You can improve our code, documentations, or even our indentations in code! We love readable code and Documentation. Pull request are welcomed.

You can even join our organisation! Feel Free to ask!

## IMPORTANT!

Comments in the code, if found to violate the Code of Conduct, will not be accepted! We will credit you, but you will not appear in the official "Contributors" Page on github. Either Code with somewhat safe comments, or don't even comment. We can add comments later.
